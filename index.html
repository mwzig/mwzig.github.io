Portfolio

Virtual Pets Amok
    Technologies/Tools Used:
  • - Java class and interface design
  • - Inheritance and polymorphism
  • - MVC, Java Map & HashMap
  • - JUnit testing

This is a fairly complex program involving organic and robotic dogs and cats with generic parent classes. The shelter houses all types of animals, and there are methods to feed and water animals that depend on passing the generic type and using instanceof to determine which method to call.

Pizza Reviews
    Technologies/Tools Used:
  • - MVC
  • - Spring
  • - Thymeleaf
  • - HTML & CSS

This is my first full-stack application using Spring and Thymeleaf for the back-end, and HTML and CSS for the front-end. It is a many-to-one design that first lists a set of reviews, and the user can click on a review to see the details. It uses a hashMap internally to represent a database of reviews.
